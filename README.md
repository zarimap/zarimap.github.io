# ザリマップ/zarimap
こちらはザリマップのサイトデータを取り扱っております。<br>
無断複製はおやめ下さい。

# ザリマップの使用方法
直感的に操作できるように設計しています。<br>
マップのピンまたは詳細パネルの情報をクリックすると、詳細が表示されます。

# クレジット
このザリマップは熊目撃情報を共有するクママップの活動に感銘を受け、コンセプトを活かしてザリガニ版として作成したものです。<br>
<b>このザリマップはクママップの運営者及びサイトとは一切の関係はございません。</b>
